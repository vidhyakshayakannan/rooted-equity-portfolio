# Rooted Equity Portfolio

A professional portfolio website for a Public Health Consultant.

## Setup
1. Clone the repository.
2. Run `npm install`.
3. Run `npm run dev` to start the local server.

## Deployment
This project is configured for Vercel.
- Push to GitHub.
- Import project in Vercel.
- The `vercel.json` ensures client-side routing works.

## Contact Form
The contact form uses [Formspree](https://formspree.io).
- Sign up for a free Formspree account.
- Create a new project/form.
- Replace `YOUR_FORM_ID_HERE` in `src/pages/ConsultationPage.jsx` with your Formspree ID.
